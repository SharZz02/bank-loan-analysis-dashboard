
# 📊 Bank Loan Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing bank loan data to understand customer borrowing behavior, loan performance, and risk patterns. The analysis is performed using SQL for data processing and Power BI for interactive visualization.

---

## 🎯 Objectives
- Analyze total loan applications and funded amounts  
- Identify good vs bad loans based on repayment status  
- Understand customer risk patterns  
- Provide insights to support better lending decisions  

---

## 🛠️ Tools & Technologies
- SQL  
- Power BI  
- Excel  

---

## 📊 Key Features
- Interactive dashboard with key performance indicators (KPIs)  
- Loan trend analysis (monthly/yearly)  
- Customer segmentation based on income and loan purpose  
- Good vs Bad loan analysis  
- Dynamic filtering and drill-through features  

---

## 📈 Key Insights
- Mid-income customers contribute the highest number of loan applications  
- Higher loan amounts are associated with increased default risk  
- Certain loan purposes show better repayment behavior  
- Loan trends indicate variations across different time periods
